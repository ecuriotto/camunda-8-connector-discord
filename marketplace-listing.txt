Brief overview / SEO description (no line breaks or bullet points)
------------------------------------------------------------------
The Discord Outbound Connector integrates Discord into your Camunda 8 BPMN processes. Send messages to channels using a bot token, post messages via webhooks, create new channels in a guild, and manage member roles — all without leaving your process diagram. Supports rich embed messages, secret-based authentication, and configurable retries.

Features
--------
1. Send messages and embeds
   Send plain-text or rich embed messages to any Discord channel using a bot token, with support for multiple embed objects via FEEL expressions.

2. Webhook messaging
   Post messages to Discord channels via webhook URL — no bot setup required. Supports username and avatar overrides.

3. Channel management
   Create text, voice, or announcement channels in any Discord guild directly from your BPMN process.

4. Role management
   Add or remove roles from guild members programmatically, enabling dynamic permission workflows triggered by process events.
